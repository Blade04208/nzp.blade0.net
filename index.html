Welcome to Blade0's Custom Map Server! This is a custom dedicated server for the game Nazi Zombies: Portable. This server was created out of the need for a dedicated server myself and friends to play each others' custom maps on. The server is open and available for anyone to join, and submit maps to.
This server runs my server, Londoneye, on Ubuntu 24.04 under Pterodactyl. Currently, the server is only available for standalone FTE clients (which is just Linux and Windows). A web server is coming soon.

For instructions on how to join, see the Joining section at the main menu.

The server map list is coming soon! For now, you can browse the map library by entering maps in the console while connected to the server, or by browsing the maps on the Github.

You can join the server using any FTE client, in 3 ways.

1. Open the Multiplayer menu, click "Browse Servers", and select "[UK] Blade0's Custom Map Server" to join.

2. Open the Multiplayer menu, click "Join IP/Code", and enter nzp.blade0.net. There is no password required.

3. Open the console with (usually) the "`" key and type connect nzp.blade0.net and hit enter.

If you have any issues with the server or have questions about anything, please join the Discord server.
